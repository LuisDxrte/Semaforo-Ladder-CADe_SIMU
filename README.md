Controle de Semáforo em Ladder (CADeSIMU)

Este projeto foi desenvolvido durante o curso técnico de automação para simular o controle de um cruzamento de semáforos utilizando lógica Ladder.

### 🛠️ Ferramentas utilizadas:
* Software CADeSIMU
* Lógica de contatos (Ladder)
* Temporizadores (TON e TOF)
